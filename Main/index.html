AWS 3-Tier Web Architecture (High Availability)

Architecture Diagram

Custom VPC with Public & Private Subnets across Multiple Availability Zones

Architecture Components

Tech Stack: AWS VPC, EC2, RDS, ALB, ASG, S3, IAM, CloudWatch, CloudTrail, SNS, NAT Gateway

Networking Layer

  • VPC: Custom VPC with CIDR block, Internet Gateway, NAT Gateway
  • Public Subnets (Web Tier): 2 subnets across AZ-1a, AZ-1b for high availability
  • Private Subnets (App Tier): 2 subnets across AZ-1a, AZ-1b for backend servers
  • Private Subnets (DB Tier): 2 subnets across AZ-1a, AZ-1b for database
  • Route Tables: Configured for public and private traffic routing
  • VPC Flow Logs: Stored in S3 bucket for network monitoring

Compute Layer

  • Web Tier (Public): EC2 instances with Nginx, serving React.js, behind External ALB with ASG
  • App Tier (Private): EC2 instances with Node.js, behind Internal ALB with ASG
  • Auto Scaling Groups: Configured for Web and App tiers with health checks
  • Load Balancers: External ALB (Internet-facing) and Internal ALB (Private)

Database Layer

  • Database Tier (Private): Amazon Aurora MySQL Multi-AZ deployment
  • Primary Database: Located in AZ-1a for read/write operations
  • Backup Replica: Located in AZ-1b for automatic failover
  • DB Subnet Group: Spans across multiple private subnets

Security & Monitoring

  • Layered Security Groups: Ext-LB → Web-Tier → Int-LB → App-Tier → DB-Tier
  • IAM Roles: EC2 roles with S3 read-only and SSM managed instance core
  • S3 Buckets: Code storage + VPC Flow Logs storage
  • CloudWatch Alarms: Monitoring metrics with SNS email notifications
  • CloudTrail: API activity logging for audit and compliance

Key Achievements

  • Achieved 99.9% availability with Multi-AZ deployment across 3 Availability Zones
  • Implemented end-to-end security with 5-layer Security Groups
  • Reduced latency by placing app and database in same VPC
  • Automated monitoring and alerting with CloudWatch + SNS
  • Scalable architecture with Auto Scaling Groups at Web and App tiers
